# Day 1 – Spinning Up an EC2 Instance

This task demonstrates how to provision an AWS EC2 instance using Terraform.

## What I implemented

- Configured AWS provider
- Used variables for AMI, instance type, key pair, subnet, and security group
- Launched a t3.micro EC2 instance
- Added tags to identify the instance

## Files in this task

provider.tf – AWS provider configuration  
main.tf – EC2 instance resource configuration  
variables.tf – Input variables for the infrastructure

## Goal

The goal of this task was to understand the basics of Terraform and how to provision cloud infrastructure using Infrastructure as Code.
