# Terraform Tasks

This repository contains my Terraform practice tasks and mini real-time infrastructure exercises.

I created this repo to build my hands-on skills in Terraform by working on practical tasks step by step. Instead of only learning theory, I wanted to actually provision, manage, and organize infrastructure in AWS using Terraform.

The goal of this repository is to showcase my learning journey in Infrastructure as Code (IaC), Terraform basics, state management, and real-world cloud provisioning tasks.

## What this repository includes

In this repository, I will be adding Terraform tasks such as:

- Provisioning AWS resources
- Launching EC2 instances
- Working with variables and outputs
- Terraform state management
- Writing reusable and clean Terraform code
- Organizing Terraform projects in a practical way

Each folder represents a separate task or concept that I practiced.

## Why I created this repository

I created this repository to strengthen my DevOps fundamentals through practical implementation.  
This helps me understand not just how Terraform syntax works, but also how infrastructure is managed in a real project environment.

I am using this repository as a place to document my progress, improve my cloud automation skills, and showcase my Terraform practice work.
